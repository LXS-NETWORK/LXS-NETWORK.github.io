Roadmap

Where LXS is and where it's going — what's done, in progress, and planned. The chain is live and the code is built; from here, the work is adoption: one more person mining, building, and using LXS.

✅ Done — the network is live

All of this exists today and you can use it right now:

  • Live proof-of-work chain with a public node, self-adjusting difficulty, halving, and fee burn.
  • Public RPC endpoint so wallets and apps can connect (and LXS adds to MetaMask).
  • One-click token launchpad — create a token that's instantly tradeable on a bonding curve.
  • Miner apps for macOS, Windows, and Linux — paste an address (or make a wallet) and mine.
  • Mining pool — steady, proportional payouts for normal computers, at 0% fee.
  • Non-custodial wallet that runs in the browser and installs to your phone's home screen.
  • Free-gas faucet so a brand-new wallet can make its first move.
  • Open-source code and a resilient peer-discovery layer so the network isn't tied to any one server.

🔨 In progress — growing the network

What we're focused on now:

  • More miners. Getting real people mining LXS — solo and in the pool — so blocks are produced by a wide, healthy set of machines.
  • A second seed node for extra resilience, so discovery never depends on a single point.
  • Community. Sharing LXS publicly and welcoming the first wave of miners, builders, and token creators.
Where we are: the technology is built and tested — the focus now is reach and adoption: getting more people mining and building on LXS.

🗓️ Planned — where we're headed

Directions we intend to take as the community grows (no fixed dates):

  • A way to buy LXS with money — a bridge and a liquidity pool on a major chain, so people can get LXS without mining.
  • More client apps — a browser extension wallet and an Android app, alongside today's web wallet.
  • Wider reach — more places to see, hold, and trade LXS as adoption justifies it.
  • More community ownership — the more people run and build on LXS, the less any single person matters. That's the goal.

Want to move the roadmap forward?

The fastest way to help is to use it: mine a block, launch a token, run a node, or read the code. LXS grows one real person at a time — and you can be one of them today.
